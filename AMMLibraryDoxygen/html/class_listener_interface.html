Detailed Description

Common listeners that can be implemented.

To receive data from subscribed topics, an object must implement this interface and override one of the methods.

Note
This interface is already implemented on DDS Manager where helper methods facilitate the exchange of data. If a user doesn't wish to use DDS Manager (which is recommended) then this interface must be manually implemented.
