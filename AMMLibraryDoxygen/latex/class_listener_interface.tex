\doxysubsection{Detailed Description}
Common listeners that can be implemented.

To receive data from subscribed topics, an object must implement this interface and override one of the methods. \begin{DoxyNote}{Note}
This interface is already implemented on D\+DS Manager where helper methods facilitate the exchange of data. If a user doesn\textquotesingle{}t wish to use D\+DS Manager (which is recommended) then this interface must be manually implemented. 
\end{DoxyNote}
