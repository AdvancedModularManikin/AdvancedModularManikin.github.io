◆ EnterListeningMode()

int EnterListeningMode ( AMM::DDSManager< void > *  mgr)

Activate listening mode and display all incomming data.

Initializes ALL topic subscribers and displays their incomming data.

Note
Intended for unit testing and controlled diagnostics. Listening to a fully operational sim will result in lots os spam and noisey data.
Parameters
Instanceof DDS Manager to initialize subscribers.
